
Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/GarudaSeva/live-stock-detection-animals-frontend.git

# Step 2: Navigate to the project directory.
cd live-stock-detection-animals-frontend

# Step 3: Install the necessary dependencies.
npm i

npm run dev
```
